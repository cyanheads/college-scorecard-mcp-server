# 0.1.0 — 2026-05-25

## Added

- **`ScorecardService`**: College Scorecard API client (`api.data.gov/ed/collegescorecard/v1`) with retry, field pre-selection, and pagination.
- **9 tools**:
  - `scorecard_search_schools` — search institutions by name, location, type, size, and acceptance rate range
  - `scorecard_get_school` — full institutional profile for one or more school IDs (costs, admissions, outcomes, aid, demographics, completion)
  - `scorecard_compare_schools` — normalized side-by-side comparison of 2–5 schools on a named topic with percentile ranks and relative deltas
  - `scorecard_get_programs` — program-level earnings (P25/median/P75), debt, and enrollment for one school
  - `scorecard_search_programs` — cross-institution program search by CIP code or keyword, ranked by median earnings
  - `scorecard_get_earnings` — institution-level earnings at 6, 8, and 10 years after entry with optional gender breakdown
  - `scorecard_value_analysis` — workflow tool: parallel-fetches cost and earnings data, computes debt-to-earnings ratio and net price to first-year earnings with peer comparisons
  - `scorecard_lookup_cip` — keyword search over the embedded CIP code taxonomy (~2,400 codes, static — no API call)
  - `scorecard_list_fields` — keyword search over the embedded field catalog (~2,800 entries, static — no API call)
- **2 resources**: `scorecard://school/{id}`, `scorecard://programs/{id}`
- **1 prompt**: `scorecard_compare_prompt` — structures a multi-school comparison analysis
- **FERPA suppression handling**: `suppressed: true` flag and `suppression_note` on program/earnings records where cohort size triggers privacy protection, preventing bare null from reaching callers
- **Embedded static data**: CIP code taxonomy and Scorecard field catalog bundled at build time; `scorecard_lookup_cip` and `scorecard_list_fields` make zero API calls
