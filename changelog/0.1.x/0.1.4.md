# 0.1.4 — 2026-05-26

- **`package.json` `bugs`** and **`homepage`** fields.

## Changed

- **`package.json` scripts** — all script commands use `bun run scripts/<name>.ts` pattern (was `tsx scripts/<name>.ts`); `start:stdio` / `start:http` use `bun ./dist/index.js` (was `node`); `bundle` uses `bun run build` (was `npm run build`); `test` uses `bunx vitest run`.
- **Dockerfile build stage** — switched from `node:24-slim` to `oven/bun:1.3` so `bun run build` resolves correctly; added `org.opencontainers.image.source` OCI label.
- **`manifest.json` version** — corrected to `0.1.3` (was `0.1.2`); now `0.1.4`.
- **`tsx` devDependency** removed — no longer needed after build scripts switched to `bun run`.
