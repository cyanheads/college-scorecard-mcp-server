# 0.1.1 — 2026-05-25

## Fixed

- **CIP code filtering**: strip the dot separator from CIP codes before sending to the API (`11.0701` → `110701`) — the Scorecard API rejects dotted codes; `scorecard_search_schools`, `scorecard_get_programs`, and `scorecard_search_programs` were all affected.
- **Repayment rate scaling**: `latest.repayment.3_yr_repayment.overall` is returned as a scaled integer (e.g. `576` = 57.6%) — divide by 1000 before use. Fixed in `scorecard_get_school`, `scorecard_compare_schools`, and `scorecard_value_analysis`.
- **Credential level field path**: `RawProgram.credential_level` → `credential.level` to match the actual API response shape; fixes `scorecard_get_programs` filtering and `scorecard://programs/{id}` resource output.
- **`format()` output in `scorecard_get_earnings`**: removed internal field path names from rendered text (e.g. `earnings_6yr_median`) — labels now read as "6-Year Median", "6-Year P25/P75", etc.
- **`format()` output in `scorecard_compare_schools`**: rank display changed from `value: X rank: Y` to `X (#Y)` for readability.
